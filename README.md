# Quick Track

Quick Track is a simple web application for tracking activities with a stopwatch feature. It allows users to log their activities, recording the start and end times, and displaying a history of their activities.

## Features

- **Stopwatch**: Start and stop a timer to track activity duration.
- **Activity Log**: View a history of logged activities with their durations.
- **Responsive Design**: Optimized for various screen sizes, making it suitable for desktop and mobile use.
